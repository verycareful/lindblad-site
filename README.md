# Q++ Website

A multi-page React and TypeScript rebuild of the Lindblad / Q++ marketing site.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

- `index.html` - overview
- `algorithms.html` - algorithm catalogue
- `benchmarks.html` - performance charts
- `contact.html` - inquiry form
- `license.html` - license summary

## Notes

- The source app lives in `src/`.
- `dist/` is generated by Vite and should not be committed.
- `node_modules/` is installed locally and should not be committed.
